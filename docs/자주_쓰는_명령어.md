# uv 명령어

### uv 가상환경 실행

```bash
source .venv/bin/activate
deactivate # 비활성화
```

### uv 패키지 설치

```bash
uv add <패키지 이름>
```

# Gemini CLI 명령어

### base-chat-session에서 시작하기

```bash
gemini --resume base-chat-session
```

### 대화 저장하기

대화 진행: "FastAPI로 유저 인증 API를 짜줘." --> 이 대화가 어느 정도 진행되면 따로 저장

```bash
/chat save feature-auth
```

### 이전에 저장된 대화 세션 목록 확인

```bash
gemini --list-sessions` # 터미널
/chat list` # 대화 중
```
